## 💡 About The Project

This is my personal portfolio website designed to present my frontend development skills to recruiters and collaborators. It makes extensive use of **Bootstrap 5 components** — including cards, badges, alerts, tabs, and forms — to create a clean, responsive, and interactive experience across all devices.

---

## ✨ Key Features

- **Fully Responsive** — Mobile-first layout across all screen sizes
- **Project Carousel** — Showcases projects using Owl Carousel
- **Smooth Animations** — Entrance animations powered by Animate.css
- **Contact Form** — Integrated with Google Sheets via Google Apps Script
- **Floating WhatsApp Button** — Quick contact access on all pages
- **Interactive Portfolio Section** — Filterable project showcase
- **Email Link** — Direct communication from the footer

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5, CSS3, JavaScript | Core structure, styling & interactivity |
| Bootstrap 5 | Responsive layout & UI components |
| Owl Carousel | Project showcase carousel |
| Animate.css | Scroll & entrance animations |
| Font Awesome 7 | Icon library |
| Google Apps Script | Contact form & Google Sheets integration |
| GitHub Pages | Web hosting & deployment |
